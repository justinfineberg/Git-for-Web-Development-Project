Fork project
Clone the URL 
Go into terminal and git clone where you want the file to exist
Git checkout to create a branch. 
Edit your code. 
Git add . 
Git commit -m "with message"
Git push -u origin name. 
Back on GitHub, click into the branch. 
Click pull request. 
Change the base repo to master and compare to branch. 
Create pull request.
